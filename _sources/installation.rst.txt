Installation
============

.. toctree::
   :maxdepth: 3

Official package (Not available yet) 
____________________________________

To install the PyPI package run the following command -

:code:`pip install textractor`

From Source
___________

To install the package, clone the repository with the following command -

:code:`git clone ssh://git.amazon.com/pkg/Aws-textract-textractor`

Navigate into the Aws-textract-textractor directory on the terminal and run these commands.

To install requirements :code:`pip install -r requirements.txt`

Then install the package with :code:`pip install .`
