Document and Page methods

This example covers the use of some methods in the Document and Page class. All methods in the Document class are also available at the Page level. Page is an entity of the Document.

Installation

To begin, install the amazon-textract-textractor package using pip.

pip install amazon-textract-textractor

There are various sets of dependencies available to tailor your installation to your use case. The base package will have sensible default, but you may want to install the PDF extra dependencies if you workflow uses PDFs with pip install amazon-textract-textractor[pdf]. You can read more on extra dependencies in the documentation

Retrieving key-values and checkboxes by geometry using directional_finder()

It is possible to retrieve key-values and checkboxes present in a certain location of the document using reference words and direction of the required key-values and checkboxes with respect to them.

[22]:

page = document.page(0)

[23]:

page.directional_finder(
    word_1="employee benefits and liability",
    word_2="primary location and subsidiaries",
    prefix ="",
    direction=Direction.BELOW,
    entities=[DirectionalFinderType.KEY_VALUE_SET],
    edit_original=False
)
